# 경쟁 검색 흡수 전략 — 합법·효과적인 방법만

## 목적
"홈페이지 제작 업체 X" 같은 검색이나 인기 서비스의 관련 키워드를 검색했을 때 **주노소프트도 함께 노출**되도록 만드는 전략. 단, 페널티/법적 리스크가 있는 방법은 모두 제외.

---

## ❌ 절대 하지 말 것

| 금지 행위 | 리스크 |
|---|---|
| 경쟁사 **상표명을 메타태그·숨겨진 텍스트·키워드 광고**에 삽입 | 상표법 침해(대법원 2015도1185), 부정경쟁방지법, 구글 cloaking 페널티 → 색인 삭제 |
| 경쟁사 **도메인 사칭** (오타 도메인 포함) | 도메인 분쟁(한국인터넷진흥원 분쟁조정) |
| **경쟁사 직접 비방** ("X사보다 절대 싸고 정직") | 표시광고법 제3조 위반(부당비교광고), 과징금 |
| 동일 콘텐츠 대량 복제 후 키워드만 바꾼 doorway page | 구글 코어 업데이트에서 수동 페널티 |
| 가짜 리뷰·바이럴 작업 | 자율심의기구·공정위 제재, 신뢰 붕괴 |

## ✅ 효과 검증된 합법 전술 (이 사이트가 이미 지원)

### 전술 1. 일반 카테고리 키워드 흡수
경쟁사도 우리도 같은 일반 키워드("홈페이지 제작", "쇼핑몰 만들기")로 검색되면, 검색결과 페이지에 **자연스럽게 같이 노출**됩니다. 누가 더 위에 있느냐의 게임이지 회피해야 할 게 아닙니다.

**자동화**: `tools/auto-blog/topics.txt` 에 일반 카테고리 토픽 70+ 시드. 매주 화·금 자동 발행.

### 전술 2. 연관 검색어 자동 흡수
구글이 자동완성으로 보여주는 **실제 사용자 검색어**가 가장 보장된 트래픽 소스. 한 번 핵심 키워드 입력하면 수십 개 롱테일을 얻음.

**자동화**: `/admin/keywords.php` — 키워드 입력 → Google Suggest 결과 → 체크박스로 선택 → 토픽 큐에 자동 추가 → 다음 자동 발행 때 글이 만들어짐

### 전술 3. 비교/추천형 롱테일 페이지
"홈페이지 제작 업체 비교", "쇼핑몰 만들기 추천", "병원 홈페이지 제작 비용 가이드" 같은 글은 **검색 의도가 명확**해서 전환율이 높음. 경쟁사들도 이런 글을 만들어 같은 결과 페이지에 함께 노출됨.

**자동화**: 토픽 큐에 비교/추천형 시드 30개 추가됨 (`# 비교/추천형 SEO 키워드` 섹션).

### 전술 4. 이미지 검색 점유
구글 이미지 검색에 포트폴리오 50건의 썸네일이 노출. 경쟁사 사이트와 같은 검색결과에 같이 등장.

**자동화**: `sitemap.xml` 안 `image:image` 항목으로 50건 자동 등록 (이미 적용).

### 전술 5. 업종별 랜딩 페이지
`/industry/hospital.html`, `/industry/academy.html` 등 5개. 업종+서비스 키워드 조합("학원 홈페이지 제작", "병원 홈페이지 비용")에서 강력. 경쟁사가 일반 페이지로 잡히는 자리에 우리는 더 정확한 페이지로 들어감.

**자동화**: 사이트맵 자동 등록, 자동 블로그 글에서 내부 링크.

### 전술 6. RSS·IndexNow 즉시 색인
Bing·네이버는 IndexNow로 즉시 색인. 새 글이 올라가면 경쟁사보다 빠르게 검색에 반영.

**자동화**: 모든 발행/변경 시 자동 핑.

## 🚫 광고 영역 주의사항

- **Google Ads**: "홈페이지 제작" 같은 일반 키워드는 OK. 경쟁사 등록상표명 입찰은 한국에서 분쟁이 잦으니 일반 키워드만.
- **네이버 파워링크**: 같은 원칙. 경쟁사 상표명은 피하고 카테고리 키워드만.
- **Meta(인스타) 광고**: 관심사 타게팅으로 충분. 경쟁사 페이지 follower만 골라 타게팅하는 기능은 제공되지 않음.

## 📈 진행 상황 측정

1. **Google Search Console** → "성능" → 클릭/노출/CTR 추이
2. **네이버 서치어드바이저** → "검색 사용자 분석" → 신규 인입 키워드
3. **이 사이트의 자체 카운트**: 자동 발행된 블로그 글 개수 = 키워드 커버리지

신규 사이트 기준 **1–2주차 색인 시작 → 4–8주차 롱테일 노출 → 3–6개월차 본격 트래픽** 흐름이 일반적입니다. 경쟁이 강한 키워드는 **6–12개월** 걸릴 수 있습니다.
