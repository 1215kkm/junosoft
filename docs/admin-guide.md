# 관리자 콘솔 사용 가이드

## 접속 주소
```
https://junosoft.co.kr/admin/
```
(닷홈에 업로드 후 운영 도메인 기준)

## 처음 사용 시 — 비밀번호 설정 (1분)
1. `/admin/` 접속 → 자동으로 `setup.php` 화면이 뜸
2. 비밀번호 8자 이상 입력 → "비밀번호 설정" 클릭
3. 자동으로 대시보드 진입
4. 이후부터는 `setup.php` 가 자동 차단되고 `login.php` 로 들어감
5. 비밀번호는 `admin/_config.php` 에 **해시 형태로** 저장됨 (평문 아님)

## 페이지 구성
| 메뉴 | URL | 기능 |
|---|---|---|
| 대시보드 | `/admin/index.php` | 미확인 문의 수, 전체 문의 수, 등록된 포트폴리오 수, 최근 문의 6건 |
| 문의 관리 | `/admin/inquiries.php` | 전체 문의 목록 + 검색/상태 필터 |
| 문의 상세 | `/admin/inquiry.php?id=...` | 모든 입력값, 메모, 상태 변경(미확인/확인/완료), 삭제 |
| 포트폴리오 관리 | `/admin/portfolios.php` | 50건 목록, 추가/수정/삭제 |
| 포트폴리오 추가/수정 | `/admin/portfolio-edit.php` | ID·제목·업종·분류·기간·가격·썸네일·태그 입력 |
| 설정 | `/admin/settings.php` | 메일·텔레그램·사이트 URL·비밀번호 변경 |

## 문의는 어디에 저장되나?
- 신청 폼이 제출되면 `_data/inquiries/{타임스탬프}.json` 으로 저장
- 동시에 관리자 메일·신청자 자동 회신 메일·텔레그램 알림 발송
- 관리자 콘솔의 "문의 관리" 메뉴는 이 JSON 파일들을 읽어서 표시
- `_data/` 폴더는 `.htaccess`로 외부 접근 차단됨 (직접 URL로 접속 불가)

## 텔레그램 알림 설정 (3분)
1. 텔레그램에서 **@BotFather** 검색 → 대화 시작 → `/newbot` 명령어
2. 봇 이름과 사용자명(끝이 `_bot`) 입력 → **봇 토큰**이 발급됨
3. 방금 만든 내 봇과 **대화 시작**(아무 메시지나 보내기). 그룹으로 받고 싶다면 그룹에 봇 초대 후 그룹에서 메시지 한 번
4. 브라우저로 다음 주소 접속:
   ```
   https://api.telegram.org/bot<봇토큰>/getUpdates
   ```
   응답 JSON 안의 `chat.id` 숫자를 복사 (그룹은 `-` 로 시작하는 음수)
5. `/admin/settings.php` 에서:
   - **봇 토큰** 입력
   - **Chat ID** 입력
   - "설정 저장" 클릭
   - "텔레그램 테스트 메시지 보내기" 버튼으로 검증
6. 설정이 정상이면 이후 문의가 들어올 때마다 자동 알림 수신

## 포트폴리오 추가 흐름
1. 새 작품 이미지를 FTP로 업로드: `images/portfolio/p051.jpg` (4:3, 1200x900 권장)
2. `/admin/portfolios.php` → "+ 새 포트폴리오 추가"
3. 입력:
   - **ID**: `p051` (영문/숫자, 중복 불가)
   - **분류**: 홈페이지/쇼핑몰/랜딩페이지/관리자/모바일앱/리뉴얼 중 택1
   - **제목·업종·기간·가격·한줄설명·태그(쉼표 구분)** 입력
   - **썸네일 경로**: 비워두면 `images/portfolio/{ID}.jpg` 자동 적용
4. 저장 → 즉시 메인/포트폴리오 페이지에 노출

## 데이터 백업
- 문의 데이터: `_data/inquiries/*.json`
- 포트폴리오 데이터: `assets/data/portfolio.json`
- 설정: `_data/settings.json`
- 비밀번호 해시: `admin/_config.php`

위 4개 폴더/파일을 주기적으로 FTP로 다운로드해 두면 모든 데이터를 보존할 수 있습니다.

## 보안 주의사항
- `admin/` 와 `_data/` 는 `.htaccess` 로 보호되지만, **HTTPS 필수**입니다 (닷홈 SSL 적용 후 사용)
- 비밀번호는 8자 이상, 가급적 16자 이상 권장
- 동시에 여러 사람이 관리자 계정을 공유하지 마세요 (IP 차단 시 다 같이 차단됨)
- 의심 IP가 6회 이상 로그인 실패 시 자동 일시 차단

## 문의가 정상 저장되는지 빠른 점검
1. 새 시크릿 창으로 `/apply.html` 접속
2. 테스트 정보 입력 후 제출
3. `/thanks.html` 정상 표시 확인
4. `/admin/index.php` 에서 "미확인 문의" 카운트가 1 늘었는지 확인
5. 텔레그램 메시지 도착 확인
6. 등록한 관리자 메일 수신함 확인
